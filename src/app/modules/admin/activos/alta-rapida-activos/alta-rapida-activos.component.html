¿Qué es el alta rápida?

Este modo está diseñado para registrar activos rápidamente con los campos mínimos necesarios:

  • Campos requeridos: Nombre, Categoría y Ubicación
  • Campos opcionales: Serie, Marca, Modelo, Descripción
  • Ideal para herramientas, consumibles controlados e inventarios rápidos
  • El activo se crea automáticamente con estado DISPONIBLE
